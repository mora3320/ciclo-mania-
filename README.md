\# Ciclo Manía - CRUD Reparaciones (Laravel)



Aplicación web en Laravel para registrar y gestionar reparaciones de bicicletas usando CRUD:

\- Listado de reparaciones

\- Filtro de reparaciones pendientes

\- Crear Editar Eliminar

\## Requisitos

\- PHP

\- Composer

\- Node.js + npm

\- MySQL Workbench

\- Laravel Herd
